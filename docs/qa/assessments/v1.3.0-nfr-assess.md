## Security

| Feature | Status | Evidence / Notes |
|---------|--------|------------------|
| Connection pre-filter | **PASS** | CIDR allow/deny at TCP accept (before TLS). Default-deny opt-in via non-empty allow list. Reload-safe via ArcSwap. 5 e2e tests incl. midstream reload |
| Cache predictor | **PASS** | Read-only from proxy perspective; no attack surface |
| Cache Vary | **PASS** | Partitions cache by request header values; `Vary: *` anchors on URI. Operator-chosen header list is the trust boundary |
| Header routing | **PASS** | Regex compiled with `regex` crate (linear-time, ReDoS-safe). Malformed regex disables rule, surfaced in UI |
| Canary | **PASS** | Deterministic IP-based bucketing. Route ID mixed into hash prevents cross-route bucket collision |
| Forward auth | **PASS** | Fail-closed on every error path. Non-loopback `http://` warn-logs at validator. Verdict cache (opt-in) is Cookie-keyed per route, TTL-capped at 60s, respects `Cache-Control: no-store` from origin, caches only Allow verdicts |
| Mirroring | **PASS** | Fire-and-forget + 256-slot semaphore. `X-Lorica-Mirror: 1` marker. Mirror cannot affect primary |
| SWR | **PASS** | Background revalidation via `SubrequestSpawner`. Stale window bounded. `LockCtx::Drop` safety net prevents dangling permits |
| Response rewrite | **PASS** | Regex pre-compiled. Compressed responses skipped. `Content-Length` stripped correctly on mutation |
| mTLS | **PASS** | PEM validated at write. `WebPkiClientVerifier` + per-route enforcement. 496/495 proven on the wire by 5 full-stack e2e tests |

**Overall Security: PASS**

## Performance

| Feature | Status | Evidence / Notes |
|---------|--------|------------------|
| Connection pre-filter | **PASS** | O(log n) CIDR lookup per accept |
| Cache predictor | **PASS** | 16-shard LRU short-circuits cache state machine. Reduces cache-lock contention on known-bypass traffic |
| Cache Vary | **PASS** | Variance computation is O(# vary headers) per request |
| Header routing | **PASS** | Rules pre-compiled; first-match O(n) with n typically < 10 |
| Canary | **PASS** | O(1) hash per request |
| Forward auth | **PASS** | Opt-in verdict cache removes auth round-trip on hot paths (cache hit = no sub-request). Off by default for strict zero-trust |
| Mirroring | **PASS** | Deterministic sampling. 256 in-flight cap bounds memory. Mirror bodies capped by `max_body_bytes` |
| SWR | **PASS** | Stale response returns immediately. Background refresh piggy-backs on cache-lock |
| Response rewrite | **PASS** | Full-body buffering capped; overflow streams verbatim |
| mTLS | **PASS** | Verifier built once at startup. `evaluate_mtls` is O(allowlist size) |

**Overall Performance: PASS**

| Cache predictor | **PASS** | Sharded LRU; no lock contention on hot path |
| Cache Vary | **PASS** | Failure modes degrade to "no variance" rather than cache poisoning |
| Header routing | **PASS** | Malformed regex disables only the rule. Dashboard surfaces disabled state |
| Canary | **PASS** | Dangling backend IDs warn at load; the split becomes inert rather than error |
| Forward auth | **PASS** | Timeout / connect error / unexpected status all fail-closed with 503 |
| Mirroring | **PASS** | Shadow failures silently dropped; primary untouched. Semaphore prevents amplification |
| SWR | **PASS** | `debug_assert` replaced with warn log in forked `lorica-cache`. `LockCtx::Drop` in lorica-proxy releases dangling permits as `TransientError`. Deterministic in debug AND release |
| Response rewrite | **PASS** | Mutual-exclusion with cache enforced at response_filter with warn log. Oversize bodies stream verbatim |
| mTLS | **PASS** | Fail-safe fallback if verifier construction fails; drift warn at reload |

**Overall Reliability: PASS**
